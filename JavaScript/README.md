# JavaScript Tutorial

A ground-up walkthrough of JavaScript fundamentals and advanced features.
Each file is self-contained and heavily commented.

## How to run

```bash
node 01_hello_world.js
```

Requires [Node.js](https://nodejs.org/) 18+.
File 13 (`dom_basics`) is browser-only — open the accompanying HTML snippet in a browser.

## Files

| # | File | Topic |
